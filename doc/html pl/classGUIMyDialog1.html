Detailed Description

Klasa dialogowa GUI do interakcji z danymi użytkownika.

Ta klasa dziedziczy po MyDialog1 generowanym przez wxFormBuilder. Obsługuje dane wejściowe użytkownika, weryfikuje je i zarządza interakcjami dialogowymi.
