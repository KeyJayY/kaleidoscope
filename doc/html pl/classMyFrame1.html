Detailed Description

Główna klasa okna aplikacji.

Klasa ta definiuje główne okno aplikacji. Zawiera różne kontrolki do manipulowania transformacjami graficznymi, takimi jak translacja, rotacja i skalowanie, zaimplementowane za pomocą suwaków i przycisków. Zapewnia również wizualną informację zwrotną tych transformacji.
